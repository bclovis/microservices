## Signaler un Problème

### Bug Report
Pour signaler un bug, créez une issue GitHub avec:
1. **Titre**: Description courte du problème
2. **Description**: Détails complets
3. **Étapes**: Comment reproduire
4. **Environnement**: Navigateur, OS, etc.
5. **Screenshots**: Si applicable

Template:
```markdown
**Description du bug**
[Description claire du problème]

**Étapes pour reproduire**
1. Aller sur '...'
2. Cliquer sur '...'
3. Voir l'erreur

**Comportement attendu**
[Ce qui devrait se passer]

**Comportement actuel**
[Ce qui se passe réellement]

**Environnement**
- OS: [e.g. Windows 10, macOS 12]
- Navigateur: [e.g. Chrome 120, Firefox 115]
- Version: [e.g. 1.0.0]

**Screenshots**
[Si applicable]
```

## FAQ

### Questions Fréquentes

**Q: Comment créer un compte?**
A: Cliquez sur "Créer un compte" sur la page d'accueil et remplissez le formulaire.

**Q: Comment changer d'équipe?**
A: Allez dans Profil > Paramètres > Changer de couleur

**Q: J'ai oublié mon mot de passe**
A: Utilisez "Mot de passe oublié" sur la page de connexion.

**Q: Comment supprimer mon compte?**
A: Contactez support@pokemon-drafter.local

**Q: Le jeu fonctionne-t-il sur mobile?**
A: Oui, l'interface est responsive.

**Q: Puis-je avoir plusieurs comptes?**
A: Non, un utilisateur = un compte (voir CGU).

**Q: Comment fonctionne le système de points?**
A: Victoire: +10, Défaite: -10 (minimum 0).

**Q: Combien de Pokemon puis-je avoir par équipe?**
A: Maximum 6 Pokemon par équipe.

**Q: Puis-je contribuer au projet?**
A: Oui! Forkez le repo et créez une Pull Request.

**Q: Le projet est-il open source?**
A: Oui, voir le repository GitHub.
