## 2. Description du service

Pokemon Drafter est une application de duel en ligne utilisant les mécaniques de combat Pokemon basées sur les avantages de types.

## 3. Inscription et compte utilisateur

### 3.1 Création de compte
- Vous devez fournir des informations exactes et à jour
- Vous êtes responsable de la confidentialité de votre mot de passe
- Un utilisateur = un compte

### 3.2 Équipe
- Lors de l'inscription, vous choisissez une équipe (Rouge ou Bleue)
- Le changement d'équipe est possible depuis les paramètres
- L'équipe détermine l'interface utilisateur et les adversaires potentiels

## 4. Règles de jeu

### 4.1 Équipes Pokemon
- Maximum 6 Pokemon par équipe
- Minimum 1 Pokemon pour créer une équipe
- Les Pokemon proviennent de PokeAPI

### 4.2 Modes de jeu
- **Hasard**: Équipes générées aléatoirement
- **Construit**: Équipes pré-construites par les joueurs
- **Pioche**: Draft alterné de Pokemon

### 4.3 Déroulement du duel
- 90 secondes par tour maximum
- Dépassement du temps = défaite automatique
- Forfait possible à tout moment
- Victoire: +10 points, Défaite: -10 points (minimum 0)

### 4.4 Calcul des avantages
Les avantages sont calculés selon la formule:
```
F(A) = 1*(W/Y)*(W/Z) + 1*(X/Y)*(X/Z)
F(B) = 1*(Y/W)*(Y/X) + 1*(Z/W)*(Z/X)
```
Le Pokemon avec F le plus élevé gagne le tour.
